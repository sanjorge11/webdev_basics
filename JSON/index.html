-- Introduction

JSON: JavaScript Object Notation.

JSON is a syntax for storing and exchanging data.

JSON is text , written with JavaScript object notation.

Exchanging Data

When exchanging data between a browser and a server, the data can only be text.
JSON is text, and we can convert any JavaScript object into JSON, and send JSON to the server.
We can also convert any JSON received from the server into JavaScript objects.

Sending Data/Receiving Data

To send JavaScript objects to the server, you need to convert the objects into a JSON string. Only text can be send to servers via HTTP. To convert JavaScript objects to JSON, use JSON.stringify() and to conver JSON strings that are sent back from servers into JavaScript objects, use JSON.parse() .

-- Syntax and Data Types

Syntax

{ "name":"John" }

Data Types

Accepted Data Types

  • a string
  • a number
  • an object (JSON object)
  • an array
  • a boolean
  • null

Non-Accepted Data Types

  • a function
  • a date
  • undefined
